
To create your translation.

1. Create folder for your translation in Languages folder, for example, DE 
2. Translate english template
3. Don't forget to include your translation files in file language_data.xml
4. Rename translation name in SubModule.xml 
5. unit_fixes150.xml file must be included in SubModule.xml . Otherwise, most warriors will not have unique translation ids.
6. Delete BYc and BYl folders
