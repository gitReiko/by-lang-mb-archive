
Amateur translation of the Cultured Start 1.1.2 mod into Belarusian. Mod can be found at https://www.nexusmods.com/mountandblade2bannerlord/mods/5342. This translation require downloading original mod from external source.

Аматарскі пераклад мода Cultured Start 1.1.2 на беларускую мову.

Фінансавая падтрымка (можа спатрэбіцца VPN) patreon.com/Reiko651

Мод Cultured Start дазваляе абраць стартавыя ўмовы. Вы можаць пачаць у якасьці васала гаспадарства, уласьніка замка, лідара наймітаў, злачынца ці іншага персанажа. Таксама льга абраць стандартныя ўмовы пачатку.

Гэта толькі пераклад. Мод Cultured Start льга спампаваць па спасылцы https://www.nexusmods.com/mountandblade2bannerlord/mods/5342 . Гэтага мода пакуль няма ў Steam Workshop.

Для кіраваньня вялікай колькасьцю модаў раю выкарыстоўваць Novus Launcher. Знайсьці загрузьнік льга па спасылцы https://www.nexusmods.com/mountandblade2bannerlord/mods/4924 .
